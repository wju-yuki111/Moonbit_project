# Contributing to MoonMoney

First off, thank you for considering contributing to MoonMoney! It's people like you that make MoonMoney such a great tool.

## Where do I go from here?

If you've noticed a bug or have a feature request, please make sure to check the [Issues](../../issues) tab first. If your issue isn't there, go ahead and create a new one!

## Setting up your environment

1. Make sure you have the [MoonBit toolchain](https://www.moonbitlang.com/download/) installed.
2. Clone the repository: `git clone https://github.com/wju-yuki111/Moonbit_project.git`
3. Run tests to ensure everything is working: `moon test`

## Making a Pull Request

1. Fork the repository and create your branch from `main`.
2. Write tests for your changes.
3. Ensure your code passes all tests: `moon test`
4. Format your code using `moon fmt`.
5. Issue that pull request!

Thank you!
